一、项目介绍
---
基于Spring Boot框架实现的在线英语阅读分级平台系统，系统包含四种角色：管理员、学生，主要功能如下。
### 【前台功能】

1. 首页：提供系统的入口，展示推荐书籍、公告等信息。
2. 公告：用户可以查看系统发布的公告。
3. 书籍：用户可以浏览系统中提供的英语阅读书籍。
4. 个人中心：用户可以管理个人信息，查看个人阅读记录、书籍收藏等。

### 【后台功能】

1. 首页：提供管理员进入后台管理的入口。
2. 个人中心：管理员可以管理个人信息。
3. 管理员管理：添加、编辑、删除系统管理员账号。
4. 基础数据管理：维护系统中的基础数据。
5. 公告管理：发布、编辑、删除系统公告。
6. 书籍管理：添加、编辑、删除阅读书籍
7. 章节管理：对书籍进行章节管理。
8. 用户管理：管理用户的操作权限。
9. 轮播图信息：管理员可以设置首页轮播图。
